# PL/SQL Control Structures

## Objective
To write basic PL/SQL blocks that manipulate data using control structures like Cursors and Loops.

## Explanation
- **Cursors:** Used to fetch multiple rows from a database table one by one.
- **Loops:** Used alongside cursors to iterate through the fetched records and apply specific business logic (e.g., updating a balance or printing a reminder).
